# Research Assistant Chatbot

A research-oriented AI assistant built using a Large Language Model (Llama-3.3-70B via Groq) to support academic workflows.

## Features
- Smart summarization (research vs general text)
- Multi-turn question answering with references
- PDF upload and contextual chat
- Literature recommendation
- Code generation

## Requirements
- Python 3.9 or higher
- Groq API key

## Installation
pip install -r requirements.txt

## Run
python groqresearchbot.py

## Notes
This project is intended for academic and research use.
